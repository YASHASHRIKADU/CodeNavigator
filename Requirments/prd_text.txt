Product Requirement Document (PRD) Product Name CodeNavigator - Smart Learning Path Generator for Tech Careers 1. Problem Statement Students and beginner developers often struggle with: Not knowing which skills to learn first Too many learning resources scattered online Lack of a structured roadmap No clear way to track learning progress As a result, many learners waste time learning random topics without direction . The Smart Learning Path Generator solves this by providing a personalized, structured learning roadmap based on the user's career goal and current skills. 2. Target Users Primary Users College students learning programming Beginner developers Self-taught programmers Secondary Users Career switchers entering tech Bootcamp students 3. Product Goals The platform should: Help users choose a tech career path Generate a structured learning roadmap Show skill gaps Track learning progress The goal is to make learning organized and goal-oriented . 4. Core User Flows Flow 1: User Registration User opens website User creates an account User logs in User accesses dashboard Flow 2: Career Path Selection User selects a career goal Example: Frontend Developer Backend Developer Full Stack Developer Data Scientist System loads roadmap template Flow 3: Skill Assessment User selects skills they already know System analyzes skill gap Missing skills are highlighted Example Known Skills HTML CSS Missing Skills JavaScript React Flow 4: Roadmap Generation System generates a step-by-step learning roadmap . Example (Frontend) Stage 1 HTML CSS Stage 2 JavaScript Stage 3 React Stage 4 Projects Flow 5: Progress Tracking User marks skills as: Completed In progress Not started Dashboard updates progress percentage . 5. Feature List MVP Features (Version 1) These features are required for launch . 1. User Authentication Sign up Login Logout 2. Career Path Selection Users choose a career goal. Example: Frontend Developer Backend Developer Data Scientist 3. Learning Roadmap Generator The system generates a structured roadmap based on the career. 4. Skill Gap Analysis User selects known skills → system identifies missing skills. 5. Progress Tracking Users can mark skills as completed. Dashboard shows progress percentage . 6. Learning Resources Each skill includes recommended resources. Example: Documentation Tutorials Practice links 7. Dashboard Dashboard shows: Selected career path Roadmap progress Next skill to learn Future Features (Post-MVP) These are not required for V1 , but possible upgrades. AI-powered roadmap recommendations Coding practice integration Community discussion Mentor guidance Job role matching Learning streak tracking 6. Edge Cases 1. User selects all skills If a user selects all roadmap skills: System should show "Roadmap completed" 2. User skips skill assessment If user skips assessment: System generates full roadmap from beginner level . 3. Invalid login attempts System should display error: "Invalid email or password" 4. User changes career path If user switches career: System regenerates roadmap. 5. No internet / API failure Resources section should show: "Unable to load resources. Try again." 7. Non-Goals (V1) These features will not be included in V1 . AI chatbots Real-time coding editor Job portal Mentor booking system Social networking features The focus of V1 is learning roadmap generation and tracking . 8. Success Metrics The product will be considered successful if: Engagement Users complete at least 30% of roadmap skills Retention Users return multiple times to update progress Usability Users can generate a roadmap in under 1 minute Adoption At least 70% of users create a roadmap after signup 9. Technical Scope (For Your React Project) Frontend React Tailwind CSS React Router Backend (optional) Node.js / Express Database MongoDB / Firebase 10. Final Product Vision The Smart Learning Path Generator will act as a digital mentor for developers , helping them: Choose the right career Follow a structured learning path Track progress toward becoming a professional developer.
